FIND ME

FIND ME est Notre premier projet dynamique réalisé lors de votre formation chez Interface3. Le site Web a été créé avec PHP, MySql pour le backend et JavaScript, HTML et CSS pour le frontend. L'idée de "Find me" est d'aider l'utilisateur à trouver un film en répondant à trois questions. L'utilisateur recevra une sélection de films en fonction des réponses données..
